Projeto de Estacionamento

Descrição do Projeto

O projeto realizado consiste no controle de um sistema de sinalização de vagas para um estacionamento. Dessa forma, é possível determinar, segundo a preferência do indivíduo e a disponibilidade, qual será a vaga que deve ser ocupada.

Foi utilizado o Quartus18 para a realização do circuito eletrônico. Faça o download do projeto abaixo.
